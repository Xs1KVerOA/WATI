# WATI v4 — 网络安全生态位人格测试

16 角色 · 15 维度 · 42 道题 · AI+安全 · 研究方向细分 · 岗位猜测

## 访问地址

- GitHub Pages：`https://xs1kveroa.github.io/WATI/`

## 特性

- **16 种安全人格**：覆盖甲方基础设施安全、应用安全、攻防安全及乙方渗透、售前、安服、安全开发等典型岗位
- **15 个评测维度**：攻防偏好、建设偏好、技术水平、工作态度、组织身份等多维度画像
- **AI × 安全态度**：融入 AI+安全趋势问题，测量你对 AI 赋能安全的态度
- **研究方向细分**：安全研究员角色自动细分漏洞挖掘 / 二进制逆向 / 红队攻防方向
- **岗位猜测**：根据答题信号推断你当前更偏甲方还是乙方
- **逃逸选项**：甲方/乙方场景题可选「我不知道，我是臭乙方/臭甲方」
